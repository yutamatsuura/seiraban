🛠️ 組み込み例

サーバーサイド統合:

// Node.js / Express での使用例 const KanteiIntegration = require('./kantei-integration.js'); const integration = new KanteiIntegration(); app.post('/api/generate-kantei', async (req, res) => { const { year, month, day, gender } = req.body; const data = await integration.fetchExistingSystemData(year, month, day, gender); const html = integration.generateKanteiHTML(data); res.json({ html, data }); });

フロントエンド統合:

// React/Vue での使用例 const integration = new KanteiSystemIntegration(); const kanteiData = await integration.fetchExistingSystemData(1982, 10, 12, 'male'); const htmlContent = integration.generateKanteiHTML(kanteiData);

🔧 技術仕様

  • データ取得: 既存システム(kigaku-navi.com)からプロキシ経由
  • キャッシュ: 同一パラメータのリクエストを高速化
  • 出力形式: HTML/JSON/構造化データ
  • PDF対応: CSS印刷メディア対応、Puppeteer等で変換可能
  • 環境: Node.js/ブラウザ両対応
